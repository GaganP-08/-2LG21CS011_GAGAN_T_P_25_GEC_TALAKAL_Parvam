Algebraic Expression

a2 – b2 = (a – b)(a + b)
(a + b)2 = a2 + 2ab + b2
a2 + b2 = (a + b)2 – 2ab.
(a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca.
(a – b – c)2 = a2 + b2 + c2 – 2ab + 2bc – 2ca.
